| a  | b  | c  | d  |
|----|----|----|----|
| \- | \- | \- | \- |
| \- | \- | \- | +  |
| \- | \- | +  | +  |
| \- | +  | \- | \- |
| \- | +  | \- | +  |
| \- | +  | +  | +  |
| +  | +  | \- | \- |
| +  | +  | \- | +  |
| +  | +  | +  | +  |
